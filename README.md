# TODO-EXPO

Aplicativo de Lista de Tarefas (To-Do) usando React Native, Expo e AsyncStorage.

## Funcionalidades
- Adicionar, editar e remover tarefas
- Persistência local usando AsyncStorage
- Navegação entre telas com React Navigation
- Interface simples e responsiva

## Estrutura do Projeto
```
App.js                # Componente principal e navegação
screens/
  TodoList.js         # Tela principal com a lista de tarefas
  AddTodo.js          # Tela para adicionar/editar tarefas
```

## Instalação
1. Clone o repositório:
   ```sh
   git clone <url-do-repo>
   cd todo-expo
   ```
2. Instale as dependências:
   ```sh
   npm install --legacy-peer-deps
   ```

## Execução
Para rodar o app:
```sh
npx expo start
```
Abra no navegador, emulador ou dispositivo físico usando o QR code.

## Observações
- Certifique-se de usar React e React DOM na versão 18.2.0 para compatibilidade com Expo Web.
- Se ocorrerem erros de dependência, use `--legacy-peer-deps` na instalação.
